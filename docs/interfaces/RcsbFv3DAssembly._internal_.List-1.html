hashCode

  • hashCode(): number

  • Inherited from Collection.Indexed.hashCode

    • Defined in node_modules/immutable/dist/immutable.d.ts:3943

    Computes and returns the hashed identity for this Collection.

    The hashCode of a Collection is used to determine potential equality, and is used when adding this to a Set or as a key in a Map, enabling lookup via a different instance.

    const a = List([ 1, 2, 3 ]);
    const b = List([ 1, 2, 3 ]);
    assert.notStrictEqual(a, b); // different instances
    const set = Set([ a ]);
